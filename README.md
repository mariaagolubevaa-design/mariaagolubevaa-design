# Портфолио — Мария Голубева

Личный сайт-портфолио UX/UI дизайнера Марии Голубевой. Статический многостраничный сайт на чистом HTML/CSS/JS, без сборки и зависимостей — макеты сверстаны по дизайну из Figma (изображения подключаются через `figma.com/api/mcp/asset/...`).

## Структура

| Файл | Описание |
|---|---|
| [index.html](index.html) | Главная страница: хедер, обложка, сетка проектов, футер |
| [hakaton.html](hakaton.html) | Кейс «Чат-бот для незрячих и слабовидящих» — хакатон IT Продлёнка 2025, Яндекс Практикум × НРОО «Инватур» |
| [tingli.html](tingli.html) | Кейс «Tingli» — профиль-центр учащегося, Wildberries × BHSAD |
| [gamification.html](gamification.html) | Кейс «Геймификация в умном доме» — SberDevices × BHSAD |
| [mendi.html](mendi.html) | Кейс «Mendi» — приложение для отслеживания приёма лекарств, дипломный проект Яндекс Практикума (в разработке) |

Каждая страница кейса содержит секции «О проекте», «Дизайн-процесс», «Исследование», «Дизайн» и «Результат» с навигацией по якорям и переходами между кейсами (пагинация).

## Технологии

- HTML5 / CSS3 (Grid, Flexbox, кастомные свойства, анимации появления через `IntersectionObserver`)
- Без фреймворков, сборщиков и npm-зависимостей
- Шрифты подключаются через Google Fonts, изображения — через Figma asset API

## Запуск

Сайт не требует сборки — достаточно открыть `index.html` в браузере или поднять локальный сервер:

```bash
python3 -m http.server 8000
```

и перейти на `http://localhost:8000`.
