О проекте

«Слабовидящему/незрячему пользователю неудобно заполнять стандартную Я.Форму, так как вопросы идут списком»
«Вопрос должен быть один на странице и озвучиваться»

Цель

Создать удобный инструмент для проведения анкетирования незрячих и слабовидящих людей c учётом A11y

Задачи

  • Создать чат-бот для проведения анкетирования незрячих и слабовидящих людей
  • Создать расширение для анкетирования в Я.Форме
  • Возможность управлять формами через админ-панель

Исследование

*Скриншоты по A11y

Целевая аудитория

  • 1Незрячие и слабовидящие люди, проходящие программу профориентации некоммерческой аудитории
  • 2Слабовидящие сотрудники некоммерческой организации, проводящие анкетирование
  • 3Зрячие сотрудники НКО, которые формируют исходные формы с ответами

Решения по реализации

  4. Расширение для Я.Форм

  1. Доступность интерфейса

  2. Telegram-бот
